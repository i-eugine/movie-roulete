# NextJS Movie Roulette

## Installing and Running the Projects

Navigate to the `nextjs` directory, install dependencies, and run the project:

```bash
cd nextjs
npm install
npm run dev
```

Open your browser and go to `http://localhost:3000`.

## Running Tests

```bash
cd nextjs
npm run test
```
