# Angular Movie Roulette

## Installing and Running the Projects

Navigate to the `angular` directory, install dependencies, and run the project:

```bash
cd angular
npm install
npm run start
```

Open your browser and go to `http://localhost:4200`.

## Running Tests

```bash
cd angular-version
ng test
```
